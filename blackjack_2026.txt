Global Variables
* Players (List)

Game Logic
* Game setup
    * Input Number of Players
    * Input Player Names
    * Input player starting money
    * Input Number of decks to play with (with default value)
    * Create decks
    * Add decks to shoe
    * Shuffle shoe
* Game loop
    * Round start
        * For each player
            * handle_bet Make bet (bet must be > minimum)
        * Deal first dealer card (remains face down for dealer)
        * For each player
            * Deal first card, face up
        * Deal second dealer card (face up)
            * If blackjack, everyone loses
        * For each player
            * Deal second card
            * If blackjack: Pay 3:2 and continue to next player
            * Else: While stay = false loop
                * Input: Hit or stay?
                * If hit
                    * If bust
                        * Announce, set bet to 0
                    * Else
                        * Announce total value 
        * Dealer
            * While Dealer hand not hard 17, hit
        * For each player - Resolve bets
            * If player busted, continue
            * Elif dealer isbusted or player hand > dealer hand, pay 2:1
            * Elif dealer hand == player hand, push (bet goes back to their bank)
            * Else dealer hand > player hand, lose (bet goes to 0)
    * Round end
        * Empty hands
            * Dealer
                * Move cards from hand to discard pile
            * For each player
                * Move cards from hand to discard pile
        * For each player
            * If money < the minimum bet, remove them from the game
        * Option to end the game
            * Print stats (bank)
        * If cards left in shoe < cut card, then add discard pile and reshuffle
